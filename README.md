# Salesforce Tools

## Project Description
Salesforce Tools (`sfdctools`) is a Next.js web application that bundles utilities for working with Salesforce metadata, formulas, and related developer workflows.

The project includes:
1. UI tools built with React/Next.js for Salesforce-focused workflows.
2. Monaco-based editor support for richer query/formula editing experiences.
3. Internal language/query support through `@datasert/langium-sfsql`.
